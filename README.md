# AI Student Counter

A real-time AI-based student counting system using YOLO, OpenCV, and Python.

## Features
- Real-time human detection
- Student counting
- YOLO object detection
- OpenCV camera processing

## Technologies Used
- Python
- OpenCV
- YOLOv8
- Ultralytics

## Run Project

```bash
py -3.12 yolo_counter.py
